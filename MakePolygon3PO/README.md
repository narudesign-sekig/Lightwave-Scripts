## MakePoly3PO

### Overview
選択した3つのポイントから4点ポリゴンを生成する。
生成されるポリゴンが平面かつ平行四辺形になるように4点目のポイントが自動的に作成される。

### Download
[SK_MakePoly3PO.py](SK_MakePoly3PO.py)

### How To Use
3つのポイントを選択した後、スクリプトを実行する。

注意 : ポイントを選択する順番が右回りと左回りではポリゴンの表裏が変わります。
